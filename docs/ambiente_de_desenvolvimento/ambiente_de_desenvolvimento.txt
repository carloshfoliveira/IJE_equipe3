1. Linguagem de programação;

- C/C++:
  É uma linguagem de uso geral, usada tanto em baixo e alto nível em ambiente 
  acadêmico e comercial, principalmente no setor de jogos devido ao alto 
  desempenho. Permite a liberdade de escolha de paradigmas orientado a objetos e
  estruturado e possui suporte a várias bibliotecas como OpenGL, SDL, DirectX
  entre outros.

2. Compilador ou máquina virtual;

- Clang: 
  É um compilador mais rápido e usa muito menos memória que o GCC. Tem alta 
  compatibilidade com GCC, suporta varias extensões de linguagem alguns dos 
  quais não são implementadas pelo GCC, como por exemplo atributos para 
  verificar a segurança de thread e tipos de vetores estendidos.

- Máquina virtual do Lua:    
  Maquina virtual que permite interpretar a linguagem de script Lua, a qual é 
  tipada dinamicamente e interpretada a partir de bytecodes utilizada pelos 
  registradores da máquina virtual. Tem como vantagem um gerenciamento 
  automático de memória com coleta de lixo incremental, fazendo assim uma 
  linguagem ideal para configuração.

3. Linguagem de script;

- Lua:  
  Possui uma engine rápida e pequena que pode ser facilmente embutida em 
  aplicações. Lua tem uma API simples, bem documentada, que permite uma 
  integração forte com códigos em outras linguagens, é simples de entender e 
  possui bibliotecas escritas em outras linguagens. Lua hoje é a única linguagem
  de programação desenvolvida fora do primeiro mundo.

4. Editor de texto;

- vim:  
  Editor simples, open source, que pode ser instalado facilmente em máquinas 
  Linux. Tem suporte a realce de sintaxe para as linguagens C/C++ e Lua.
    
5. Depurador;

- gdb:
  O GNU Debugger (gdb) é um ferramenta de debugger usada em 
  arquivos executáveis, na qual é possível fazer o percurso de uma execução
  e identificar problemas de erros de execução como ponteiro nulo. Esta 
  ferramenta não possui interface gráfica.

6. Ferramenta de controle de versão;

- git: 
  Ferramenta de controle de versão em que é possível criar uma cópia bem mais 
  rápido que outras ferramentas de controle de versão, é otimizada para 
  funcionar na internet, seus repositórios ocupam menos espaços, é mais fácil de
  administrar diferentes fontes de modificações, permitindo um melhor fluxo de 
  trabalho.

7. API gráfica;

- SDL2:
  É uma biblioteca multimídia livre escrita em C, que permite representações 
  gráficas, som, rede e dispositivos de entrada em várias plataformas como 
  Windows, Mac OS X, Linux, iOS e  Android.

8. API de áudio;

- SDL_mixer:
  Biblioteca SDL com funções de áudio complexas, principalmente para mixagem de 
  som, permite carregamento de arquivos de áudio em formato FLAC, Ogg Vorbis, 
  SMPEG e MP3.

9. API de rede;

- SDL_net:
  API que permite criação de um chat cliente e um servidor de aplicação.

10. Analisador de código dinâmico;
- valgrind:   
  Framework para construção de ferramentas de analise dinâmica. O valgrind 
  possui ferramentas que podem detectar automaticamente erros de gerenciamento 
  de memória, dois detectores de erros em thread, cache, branch-prediction, 
  profile, gerador de gráficos e heap profile.

11. APIs para manipulação de arquivos;

- SDL_image:
  API que permite a manipulação de imagens em vários formatos (BMP, GIF, JPEG, 
  LBM, PCX, PNG, PNM, TGA, TIFF, WEBP, XCF, XPM, XV), carregando as imagens como
  uma estrutura SDL, permitindo um controle das imagens sobre a API SDL de 
  montagem de janelas.
    
12. Sistema Operacional

- Ubuntu:   
  Sistema operacional open source, desenvolvido pela comunidade, que possui uma 
  gama de programas livres e com um alto potencial para aprendizado e 
  desenvolvimento de software acadêmicos e comerciais.
